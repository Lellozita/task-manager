# Task Manager (To-Do List)

Projeto frontend simples para gestão de tarefas, desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas da lista

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
